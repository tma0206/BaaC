# トラブルシューティング（Debugging Guide）

**Document:** `docs/troubleshooting.md`  
**対象:** 葉の変色・枯れ込み・害虫など **Bug** 相当の観測に対する切り分け

---

## 観測ログの取り方（再現手順の共有）

Issue を立てる前に、以下をメモすると解決が早いです。

1. **いつから** 気づいたか（デプロイ／環境変更の直後か）。
2. **どの枝・どの面** か（写真 1〜3 枚）。
3. 直近の **Update**（水やり）頻度と量。
4. 置き場所（日当たり、風、室内／屋外）。

---

## 1. 葉の変色（Bug: Discoloration）

| 症状 | よある原因 | まず試すこと |
|------|------------|----------------|
| 全体が薄黄化 | 過湿・根の呼吸不良 | Update 間隔を延ばし、排水と風通しを確認 |
| 先端から茶枯れ | 乾燥しすぎ、肥害、または一時的な日焼け | 日陰に移し、Update を少量・回数多めに |
| 斑点状 | 真菌性の可能性、または葉上の付着物 | 該当葉を隔離観察、専門家または農薬適用はラベルに従う |
| 一部だけ急激に茶色 | 害虫・線虫・根の局所死など | 拡大防止のため該当枝を写真に撮り、植木屋・診断サービスへ |

**切り分けのコツ:** 「新しい枝だけか」「内側の老葉だけか」で、ログの優先度が変わる。

---

## 2. 落葉・大量ドロップ（Bug: Unexpected Release）

- **季節のリリースノート:** 真柏は一部の古葉の自然更新あり。全体的でなければパッチレベル。
- **環境変更直後:** 移動・室内取り込み直後は一時的なストレス反応のことがある。極端な場所変更を避ける。

---

## 3. 害虫（Bug: External Dependency Injection）

| 兆候 | 候補 |
|------|------|
| 白い綿・モワモワ | カイガラムシ系 |
| 葉がギラつく | アブラムシ系 |
| 葉が薄くなる・蜘蛛の巣様 | ハダニ系 |

**対応:** まず水洗いやアルコール綿棒など物理的な **ホットフィックス**。広がる場合は適正農薬のラベルに従う（自己責任で安全に）。

---

## 4. エスカレーション

以下は **Issue テンプレート（トラブル報告）** からエスカレーション推奨。

- 幹や根元の軟化・黒変（クリティカル）
- 2 週間以上改善しない全体の黄化
- 害虫が他の盆栽へ横展開している
